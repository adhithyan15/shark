# Shark

Shark is a simple testing tool for file operations. 

## Installation

Add this line to your application's Gemfile:

    gem 'shark'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install shark

## Usage

It has a cucumber like syntax and is intended to be used to test file operations.
